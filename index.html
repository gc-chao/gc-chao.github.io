2018-2021 | Transition to Architecture Design & Project Management

As more government systems were developed, my role gradually shifted from front-line programmer to architecture design and project manager. I refactored the development framework to adapt to SOA architecture and introduced numerous cloud service middleware components. To address diverse requirements, I expanded the framework with script, workflow, and low-code engines, along with support for WebSocket, SIP, ModBus, GIS, and streaming media technologies. Additionally, following software engineering principles, I established a comprehensive set of development standards and management systems, and led the team to build an internal collaboration platform.

  • Platforms: Windows, Linux, Android, iOS, Cloud Services
  • Languages: C/C++, Java, Kotlin, Objective-C, JavaScript
  • Frameworks & SDKs: Win32, POSIX API, Winsock, JDK, Android SDK, Cocoa
  • Databases: PostgreSQL, Oracle, MongoDB, SQLite
  • Tools & Technologies: JNI, Nginx, FFmpeg, ArcGIS

2015-2018 | Embracing Mobile Internet & Passion for Building Wheels

Recognizing Java's potential for future internet development, I shifted the web stack to JSP/Servlet. From this foundation, I built basic scaffolding that separated request routing, controller logic, and view rendering. Through practical projects, I refined it by adding ORM, MVC, AOP, MQ, and other components, eventually evolving it into an internal backend framework. Similarly, what started as a simple AJAX-based SPA rendering engine evolved into an internal frontend framework with MVC, animation, UI, and native-bridge features. As I embraced the surging mobile internet, I tackled new architectural challenges, including payment security, concurrency and synchronization, load balancing, caching strategies, and third-party service integration.

  • Platforms: Linux, Android, iOS, WeChat
  • Languages: Java, JavaScript, Objective-C
  • Frameworks & SDKs: JDK 1.6/1.8, Android SDK, Cocoa
  • Databases: MySQL 5.x, MongoDB, SQLite
  • Tools & Technologies: Eclipse, Android Studio, Xcode, Apache, Tomcat, SVN, Markdown
